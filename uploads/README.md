# Uploads

Esta carpeta contiene archivos subidos dinámicamente por la aplicación.

## Estructura

- **fighters/** - Fotografías de los peleadores

## Nota importante

Las imágenes en esta carpeta NO se suben al repositorio Git por razones de espacio y privacidad. Solo los archivos README.md se mantienen en el repositorio para documentar la estructura.

Al desplegar la aplicación en producción, asegúrate de:
1. Crear esta estructura de carpetas
2. Configurar los permisos de escritura apropiados
3. Considerar usar un servicio de almacenamiento externo (S3, Cloudinary, etc.) para producción
