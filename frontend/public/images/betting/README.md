# Betting

Esta carpeta contiene las imágenes relacionadas con las apuestas:

- Iconos de tipos de apuestas
- Chips o fichas
- Gráficos de victoria/derrota
- Iconos de categorías de peleas (Main Card, Prelims, etc.)
- Elementos visuales para las opciones de apuesta

**Formatos recomendados:**
- PNG transparente o SVG para iconos
- Tamaños consistentes para mejor presentación
