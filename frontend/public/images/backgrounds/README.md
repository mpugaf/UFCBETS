# Backgrounds

Esta carpeta contiene imágenes de fondo para diferentes secciones:

- Fondo de página de inicio/login
- Fondos para secciones específicas
- Patrones o texturas
- Imágenes de hero sections

**Formatos recomendados:**
- JPG para fotografías (optimizadas para web)
- PNG para fondos con transparencia
- WebP para mejor compresión
