# Logo

Esta carpeta contiene las imágenes del logo de la aplicación:

- Logo principal en diferentes formatos (PNG, SVG)
- Favicon
- Logo para diferentes tamaños y contextos (header, login, etc.)
- Versiones en blanco/oscuro si aplica

**Formatos recomendados:**
- Logo principal: SVG o PNG transparente
- Favicon: ICO, PNG (16x16, 32x32, 180x180)
