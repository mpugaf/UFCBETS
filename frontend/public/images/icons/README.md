# Icons

Esta carpeta contiene iconos generales de la interfaz:

- Iconos de navegación
- Iconos de acciones (editar, eliminar, ver, etc.)
- Iconos de estado (éxito, error, advertencia, info)
- Iconos sociales
- Otros iconos de UI

**Formatos recomendados:**
- SVG para escalabilidad
- PNG transparente como alternativa
