Stack Tecnológico Recomendado: Node.js + React
Por qué esta combinación:
Backend: Node.js + Express

Ecosistema maduro y gratuito
Excelente para APIs REST
Compatible con MySQL (mysql2, Sequelize)
Deploy gratuito: Railway, Render, Fly.io
Manejo eficiente de conexiones concurrentes

Frontend: React

Ventaja clave: React Native permite reutilizar código para app móvil nativa después
Comunidad más grande que Angular/Vue
Más fácil encontrar soluciones y librerías
PWA capabilities para web móvil
Deploy gratuito: Vercel, Netlify, Cloudflare Pages

Descartando alternativas:

❌ Angular: Más pesado, curva de aprendizaje mayor
❌ Vue: Menor ecosistema, menos recursos para móvil
